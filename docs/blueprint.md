# **App Name**: EF Developers

## Core Features:

- Property Listings: Display a comprehensive list of available properties, including images, descriptions, and pricing.
- Property Details: Provide detailed information about each property, including floor plans, amenities, and location details with the aid of interactive maps.
- AI-Powered Property Recommendations: Use a generative AI tool to analyze user preferences and property data to recommend suitable properties.
- New Launches Section: Feature newly launched properties with a 'Tracing Beam' effect on the PropertyCard to highlight them.
- About Section: Share the brand's history since 1971, location in Limassol, Cyprus, and showcase the values that represent the brand.
- Contact Information: Enable potential clients to easily contact the developers.

## Style Guidelines:

- Primary color: Gold (#D4AF37) for a prestigious and exclusive feel, paired with dark backgrounds for contrast.
- Background: Dark shades (#0a0a0a, #121212, #1a1a1a) to emphasize luxury and timelessness.
- Accent color: A slightly lighter gold (#E5C565), for a soft gleam.
- Headings: 'Playfair Display', serif, for main titles, property names.
- Body: 'Manrope', sans-serif, for paragraphs, UI elements, labels.
- Accents: 'Cormorant Garamond', serif, for italicized quotes and luxury sub-labels.
- Asymmetric Bento Grid layout strategy, container padding for spaciousness, and a maximum width to maintain focus.
- Smooth scrolling with Lenis for a luxury feel; micro-animations on every interaction; default transition: all 0.5s cubic-bezier(0.22, 1, 0.36, 1)
